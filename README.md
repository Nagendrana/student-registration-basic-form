# 🎓 Student Registration Form

A simple and user-friendly **Student Registration Form** built with HTML, CSS, and JavaScript.  
This project is designed to collect and manage student details efficiently, making it suitable for schools, colleges, and online learning platforms.

---

## ✨ Features
- 📋 Input fields for **Name, Age, Gender, Email, Phone Number, and Address**
- 🎯 Dropdowns for **Course selection** and **Department**
- ✅ Form validation to ensure accurate data entry
- 📱 Responsive design for desktop and mobile devices
- 🖱️ Clear layout with labels and instructions
- 💾 Submit button to store or process student information

---

## 🛠️ Technologies Used
- **HTML5** → Structure of the form  
- **CSS3** → Styling and layout  
- **JavaScript** → Validation and interactivity  

---

## 🎯 Purpose
This form helps institutions **digitize student enrollment**, reduce paperwork, and maintain a structured database of student records.  
It can be customized to fit specific academic requirements.
